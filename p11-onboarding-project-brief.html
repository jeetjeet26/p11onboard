Project Overview

New Client Onboarding System

Bri (VP of Digital Marketing) is building a streamlined onboarding system for P11creative that reduces manual work, gives clients a clear progress experience (Dominos-style tracker), and integrates with Accelo, Basecamp, Dropbox, Google Drive, and Softr via automation.

Phase 1 Complete: Dashboard + Automation Blueprint Up Next: Automation Build + Team Review
Deliverables Built This Session
🖥️
p11-onboarding-dashboard.html
Fully branded interactive onboarding portal. Toggle between Client View (Dominos-style progress tracker, action items, timeline, and full intake questionnaire) and Team View (stats snapshot, team assignments, quick links, and a full tabbed task checklist by service). Built with P11creative brand: charcoal base, P11 Red #FA4614 accents, Inter font.
⚡
p11-onboarding-automation-flow.html
Full Zapier / Make.com automation blueprint. Covers 4 Zaps: (1) Master Intake Handler — 12-step automation that fires on form submission to create Accelo records, Basecamp project, Dropbox folders, Google Drive files, Softr portal, kickoff email, and Slack notification. (2) 48-hour access reminder sequence. (3) Ad copy approval loop. (4) Launch day automation. Includes prioritized build order and implementation notes.
